UazAPI — envio e instância

Envio: POST {base}/send/text?token=... (documentação docs.uazapi.com). Base URL costuma ser https://seudominio.uazapi.com (ex.: https://focus.uazapi.com). Tokens gravados cifrados.

Provedor efetivo: — · WHATSAPP_PROVIDER=auto|uazapi|meta no ambiente.

Legado — Meta Cloud API (webhook /whatsapp)
